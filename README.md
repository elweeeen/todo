# TODO アプリ

インストール・ビルド・サーバー不要の、シンプルな TODO 管理アプリです。
素の HTML / CSS / JavaScript のみで動作し、データはブラウザの localStorage に保存されます。

## 使い方

`index.html` をダブルクリックしてブラウザで開くだけです。

## 機能

- ✅ タスクの追加・完了/未完了の切り替え・削除
- ✏️ ダブルクリック（または ✎ ボタン）でインライン編集
- 🚩 優先度（高 / 中 / 低）の設定と色分け表示
- 📅 期限の設定と、期限切れタスクの強調表示
- 🔍 フィルタ（すべて / 未完了 / 完了）
- 🧹 完了済みタスクの一括削除
- 💾 localStorage による自動保存（閉じても消えません）

## 並び順

未完了 → 優先度（高い順）→ 期限（近い順）→ 追加順 で自動的に並びます。

## ファイル構成

| ファイル | 役割 |
|----------|------|
| `index.html` | 画面の構造 |
| `styles.css` | デザイン |
| `app.js` | 動作ロジック（追加・編集・保存など） |

## データの保存場所

ブラウザごとの localStorage（キー: `todo-app:tasks:v1`）に保存されます。
別のブラウザや別の端末とは共有されません。
